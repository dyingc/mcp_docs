# Document Index
- [Bug Report Template](https://raw.githubusercontent.com/modelcontextprotocol/java-sdk/main/.github/ISSUE_TEMPLATE/bug_report.md): This document outlines a template for reporting bugs, including sections for bug description, environment details, steps to reproduce, expected behavior, and a minimal complete reproducible example. It emphasizes checking for existing issues and providing clear, detailed information for efficient resolution.
- [Feature Request Form](https://raw.githubusercontent.com/modelcontextprotocol/java-sdk/main/.github/ISSUE_TEMPLATE/feature_request.md): This form allows users to submit feature requests for a project. It prompts users to describe the expected and current behavior, provide context, and check existing GitHub issues before submission.
- [Miscellaneous](https://raw.githubusercontent.com/modelcontextprotocol/java-sdk/main/.github/ISSUE_TEMPLATE/miscellaneous.md): This document outlines the guidelines for filing issues and requests related to the project. It clarifies that the issue tracker is exclusively for bug reports and feature requests, and directs users with questions or support needs to GitHub Discussions.
- [Contributor Covenant Code of Conduct](https://raw.githubusercontent.com/modelcontextprotocol/java-sdk/main/CODE_OF_CONDUCT.md): This Code of Conduct outlines pledges and standards for community participation, emphasizing a harassment-free and inclusive environment. It details acceptable and unacceptable behaviors, enforcement responsibilities of community leaders, and the scope of the code. It also provides enforcement guidelines ranging from correction to a permanent ban for violations.
- [Contributing to Model Context Protocol Java SDK](https://raw.githubusercontent.com/modelcontextprotocol/java-sdk/main/CONTRIBUTING.md): This document outlines how to contribute to the Model Context Protocol Java SDK, covering prerequisites, getting started, reporting issues, making and submitting changes, and adherence to the Code of Conduct and licensing.
- [MCP Java SDK](https://raw.githubusercontent.com/modelcontextprotocol/java-sdk/main/README.md): The MCP Java SDK provides a standardized interface for Java applications to interact with AI models and tools, supporting both synchronous and asynchronous communication patterns. It offers comprehensive documentation and integrates with Spring AI for easy bootstrapping of AI applications. The project is open for contributions and licensed under the MIT License.
- [Security Policy](https://raw.githubusercontent.com/modelcontextprotocol/java-sdk/main/SECURITY.md): This document outlines Anthropic's security policy for their SDKs and related systems, emphasizing responsible disclosure through their HackerOne vulnerability disclosure program. It provides clear instructions for reporting security issues to Anthropic via their official HackerOne submission form.
- [WebFlux SSE Transport](https://raw.githubusercontent.com/modelcontextprotocol/java-sdk/main/mcp-spring/mcp-spring-webflux/README.md): This document describes how to use the WebFlux SSE Transport library, including adding the dependency, configuring the SSE transport bean, and setting up the MCP server with various capabilities like resource, tool, and prompt support.
- [WebMVC SSE Server Transport](https://raw.githubusercontent.com/modelcontextprotocol/java-sdk/main/mcp-spring/mcp-spring-webmvc/README.md): This document describes how to configure and use the WebMVC SSE Server Transport, including its Maven dependency and a Java configuration example. It outlines the setup for a WebMvcSseServerTransport bean and a router function for handling messages.
- [Java MCP SDK](https://raw.githubusercontent.com/modelcontextprotocol/java-sdk/main/mcp/README.md): Java SDK implementation of the Model Context Protocol for seamless integration with language models and AI tools. Comprehensive guides and API documentation are available on the MCP Java SDK Reference Documentation.
- [MCP Java SDK Migration Guide: 0.7.0 to 0.8.0](https://raw.githubusercontent.com/modelcontextprotocol/java-sdk/main/migration-0.8.0.md): This document provides a guide to migrating code from version 0.7.0 to 0.8.0 of the MCP Java SDK, outlining breaking changes such as the introduction of a session-based architecture, new transport provider abstraction, exchange objects for client interaction, renamed interfaces, and updated handler signatures, along with detailed examples for various migration scenarios.
